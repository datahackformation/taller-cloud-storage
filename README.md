# Taller: Storage services en cloud (GCP, Azure, AWS)

## Ejercicios
### CLI y Python
1) Autenticación.
2) Habilitar servicios de storage.
3) Detalles de elementos en storage.
4) Eliminar elementos storage en cloud.
5) Descargar objetos desde cloud.
